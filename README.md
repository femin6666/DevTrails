# 🚀 GigSafe: AI-Powered Insurance for Delivery Partners

## 📌 Problem
Delivery partners lose income due to weather and external disruptions.

## 👤 Persona
Swiggy Delivery Partner in Chennai

## 🔄 Workflow
1. Login
2. Select city
3. AI calculates weekly premium
4. System detects disruption
5. Claim auto-triggered
6. Instant payout

## 💰 Weekly Premium
- ₹40 base
- Increases based on risk

## ⚡ Triggers
- Rain
- Pollution
- Curfew

## 🧠 AI Features
- Dynamic pricing
- Risk scoring
- Fraud detection
